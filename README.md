# HorizonTechX Portfolio Website

A modern, fully-responsive personal portfolio website built with HTML, CSS, and JavaScript. Showcase your skills, projects, resume, and contact information with style and elegance.

## ✨ Features

### 🎨 Core Features:
- ✅ **Responsive Design** - Works perfectly on all devices (desktop, tablet, mobile)
- ✅ **Modern UI** - Beautiful gradient backgrounds and glassmorphism effects
- ✅ **Smooth Scrolling** - Seamless navigation between sections
- ✅ **Animated Elements** - Engaging animations and transitions
- ✅ **Multiple Sections**:
  - Hero Section with call-to-action buttons
  - About Me with statistics
  - Skills & Technologies
  - Featured Projects showcase
  - Contact Information & Form
  - Footer with social links

### 🎁 Bonus Features:
- ✅ **Smooth Animations** - CSS animations and transitions throughout
- ✅ **Hover Effects** - Interactive elements on hover
- ✅ **Mobile Hamburger Menu** - Responsive navigation
- ✅ **Scroll to Top Button** - Quick navigation
- ✅ **Intersection Observer** - Elements animate on scroll
- ✅ **Form Validation** - Contact form with validation
- ✅ **Social Links** - Connected social media profiles
- ✅ **Counter Animation** - Animated statistics

## 🎨 Customization

### Colors
Modify the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #00d4ff;      /* Cyan */
    --secondary-color: #00ff88;    /* Green */
    --bg-dark: #0a0e27;            /* Dark background */
    --text-primary: #ffffff;       /* White text */
    --error-color: #ff3366;        /* Red */
}
```

## 🚀 Deployment

### GitHub Pages (Free)
1. Push your files to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Your site will be live at: `https://yourusername.github.io/repository-name`

### Netlify (Free)
1. Connect your GitHub repository
2. Netlify will automatically deploy on push
3. Get a custom domain or use the provided Netlify domain
